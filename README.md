# Auth0 CLI POC 

## Example Usage Windows

```bash
auth0_cli_poc.exe
     --auth0-domain <domain>
     --auth0-client-id <client-id>
     --audience <audience>
     --auth0-client-secret <Client-Secret>
     --auth0-scopes "openid profile email"
```
